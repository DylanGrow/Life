# LIFELOOP — Complete BitLife Clone

Single-file life simulator with deep careers, mortgages, businesses, 20-stock market with options, prison, fame, and three-generation family.

## Features
- Deep career trees (Fishing, Medical, Business)
- Real estate with 20% down mortgages
- Business empire (New Bern Marina, etc.)
- 20 tickers + options trading
- Crime & prison system
- Fame & social media
- Family: grandparents, parents, spouse, kids, grandkids, reunions, inheritance
- Three.js orb, D3 chart, Tone.js sounds
- Local save (no server)

## Quick Start
1. Download `lifelooop_final.html`
2. Double-click to play in browser

## Deploy to GitHub Pages
1. Create new repo on GitHub (e.g., `lifelooop`)
2. Upload `lifelooop_final.html` and rename to `index.html`
3. Upload `manifest.json` and `icons/` folder
4. Go to Settings → Pages → Deploy from main branch
5. Your game will be live at `https://yourusername.github.io/lifelooop/`

## Mobile Install (PWA)
- iOS: Open in Safari → Share → Add to Home Screen
- Android: Open in Chrome → Menu → Install app

## Files
- `index.html` - the game
- `manifest.json` - PWA config
- `icons/` - app icons
- `README.md` - this file
